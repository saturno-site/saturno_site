# Saturno Hermes local optimization

Canonical versioned Hermes project context lives here, inside the git repo.

Purpose: keep Hermes lean for this project.

## Files

- `.hermes/SESSION_START.md`: startup checklist.
- `.hermes/context.md`: compact context loaded/read at session start.
- `.hermes/memory/`: repo-specific durable memory.
- `.hermes/plans_tasks/`: Plannotator plans and task state.
- `.hermes/skills/saturno-enneagram-web/SKILL.md`: local workflow skill.

## Lean default tool posture

Keep:
- terminal, file, code_execution, web, skills, todo, memory, session_search, clarify, delegation.

Enable only when needed:
- browser, vision, image_gen, tts, messaging, cronjob, computer_use, moa.

MCP:
- Keep `morph-mcp` for codebase search/edit.

## Versioning rule

Commit compact, non-secret `.hermes/` docs/skills/memory/plans that help future sessions.
Do not commit volatile logs, sessions, caches, or secrets.
